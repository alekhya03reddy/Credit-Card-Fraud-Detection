# Credit Card Fraud Detection

This project detects fraudulent credit card transactions using Machine Learning techniques.

## Objective
To build a classification model that identifies fraudulent and genuine credit card transactions.

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

## Machine Learning Algorithm
- Logistic Regression

## Features Used
- Transaction Features
- Amount
- Time
- Encoded Variables

## Steps Performed
- Data Loading
- Data Cleaning
- Missing Value Handling
- Feature Selection
- Train-Test Splitting
- Model Training
- Fraud Prediction
- Model Evaluation
- Data Visualization

## Evaluation Metrics
- Accuracy Score
- Classification Report
- Confusion Matrix

## Dataset
Credit Card Fraud Detection Dataset from Kaggle:  
https://www.kaggle.com/datasets/kartik2112/fraud-detection

## Conclusion
Successfully built a Machine Learning model to detect fraudulent credit card transactions.
